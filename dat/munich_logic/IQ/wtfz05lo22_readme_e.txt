Note:
1. Research data files
Please consider the following data import instructions for statistic software:
* research data matrices are archived as delimited plain text files 
* the rows of the matrix correspond to the units of investigation and the columns to the variables
* the first row contains variable name.
* variables are tab-separated
* variable information and value and missing labels can be found in the codebooks

2. Codebook
A file containing metadata at the variable level, structured according to the following syntax:
* Variable name (in capital letters)
* Variable description
* Item text (if available), instruction or information how to calculate the variable
* {valid data codes}
* {missing data codes}
* Value labels
* Missing values
* Free line

3. Additional Material
Other documents that are important for the interpretation and re-use of the data, e.g. program code, sketches, concordance lists for items from longitudinal studies.

4. Data File Names
Data file names are composed of
*dataset ID
*wave (e.g. w3 = third wave)
*data file abbreviation
*consecutive number

Explanation of data file abbreviations:
_fd: Research data file
_kb_e: English codebook
_kb_g: German codebook
_iz_e: English Concordance list
_in_g: German Instructions
